## 🆘 TROUBLESHOOTING RÁPIDO

| Problema | Solución |
|----------|----------|
| Flyer no aparece | F12 → Console → Busca "Error" |
| Dice "Sin cervezas" | Google Sheets no tiene datos para viernes |
| Error "fetch timeout" | GAS_URL_FLYER es incorrecta |
| Fotos no se ven | Nombres en Sheets ≠ archivos GitHub |
| Botones no funcionan | F12 → Console → Copy error |
